Amazon oferece voz do ator Samuel L. Jackson
na assistente virtual Alexa

16/05/2019

A Amazon passará a oferecer vozes de celebridades para representar sua assistente virtual, a Alexa, como
forma de atrair mais usuários para o serviço. Para a estreia, a empresa escolheu o ator Samuel L. Jackson.
De acordo com a Amazon, o serviço custará US$ 0,99 para assinaturas feitas em 2019. A partir do ano que vem,
quando vozes de mais celebridades estarão disponíveis, o valor também deverá ficar mais alto: US$ 4,99.

GitHub lança repositórios privados gratuitos para até três colaboradores

07/06/2019

O GitHub anunciou que permitirá aos desenvolvedores criar repositórios privados para até três colaboradores,
sem nenhum custo. Além disso, o GitHub introduziu um novo produto para clientes corporativos que operam
tanto no cloud como em um data center local.

O GitHub há muito oferece contas de usuário gratuitas sob a premissa de que elas foram usadas para
repositórios públicos. Essa foi uma excelente combinação para projetos open source e permitiu que o GitHub
atraísse milhões de projetos de código aberto e merecesse o rótulo da enciclopédia mundial de software de
código aberto. Agora, com o GitHub Free, os desenvolvedores poderão criar repositórios privados e
compartilhá-los com mais dois colaboradores. Além disso, o GitHub descreve o GitHub Free como fornecendo
acesso ilimitado a repositórios privados sem nenhum custo.
